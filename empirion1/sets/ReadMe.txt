Flanker.txt

Participant: participant ID	
Block: block number	
Trial: trial number	
Stimulus: stimulus presented to the participant	
Condition: neutral, congruent, or incongruent flankers	
CorrectResp: response that should be given	
Code_CorrResp: the key that corresponds to correct response	
Response_Code: the actual key pressed	
Response_Time: time key was pressed after trial onset
